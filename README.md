## Overview

This repository contains a retail sales dataset used to create an interactive Excel dashboard. The dataset includes transaction-level sales data such as customer demographics, product information, and order performance.

The project demonstrates data analysis and visualization using Microsoft Excel.

## Dataset Contents

The dataset includes:

* Order and transaction details
* Customer gender and age group
* Order date and status
* Sales channels (Amazon, Myntra, Flipkart, etc.)
* Product category and size
* Sales amount and quantity
* City and state information

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Dashboard Visualization

This dataset is used to practice Excel-based data analysis and to build a sales dashboard for identifying trends and business insights.
